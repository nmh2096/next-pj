# Note App Auth API

### Abstract
This is API of Note App

### Prerequisites
Before you continue, ensure you meet the following requirements:
- Installing a Code editor (ex: VS Code)
- Installing Nodejs
- Install MongoDB

### Installation 
- Open Terminal
- Run `npm i`
- Run `npm start`

### Technologies
- Framework: `ExpressJS`
- Database: `mongoDB` `mongoose`
- Auth: `jsonwebtoken`
